Este repositório contém a **base de um portfólio pessoal**, desenvolvido utilizando **HTML, CSS e JavaScript**.

A ideia do projeto é servir como estrutura inicial para apresentar projetos, informações e habilidades de forma organizada em uma página web.

# ↪︎ Sobre o projeto

Este projeto foi criado como **base para um portfólio de desenvolvedor**, permitindo apresentar:

- projetos desenvolvidos
- informações pessoais
- habilidades
- contatos

A estrutura pode ser facilmente modificada e expandida para criar um **portfólio completo**.
